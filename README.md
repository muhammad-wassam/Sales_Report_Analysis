# 📊 Sales Report Analysis

## 📌 Project Description
**Sales Report Analysis** is an Excel-based project that analyzes **Vrinda Store sales data** to uncover business insights.  
The analysis uses **Pivot Tables, charts, and data summaries** to identify **sales trends, top-performing products, and customer behavior**.

## 🛠 Tools Used
- **Microsoft Excel**
- **Pivot Tables**
- **Charts & Data Visualization**

## 📈 Key Insights
- **Monthly sales trends**
- **Top-selling product categories**
- **Customer purchasing patterns**
- **Revenue distribution**

## 📂 File
- **Sales_Report_Analysis.xlsx** → Contains the dataset and analysis.
